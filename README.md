# web-console
Modular Web Console Prototype based on Angular

## Prerequisites

This setup requires a nodejs (https://nodejs.org) installed locally on your workstation.

## Install guide

```console
cd web-admin-core
npm install
npm run build

cd ..
cd tick-tock-module
npm install
npm run build

cd ..
cd WebAdminDemoApp
npm install
npm run start
```


then open browser at: http://localhost:4200/
